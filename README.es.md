## Funciones

- **Notas rápidas por sitio** — abre una hoja en cualquier sitio web y añade texto enriquecido, notas adhesivas, botones de enlace e imágenes. Arrastra imágenes directamente desde tu ordenador o desde otra página.
- **Varias páginas** — cada sitio puede tener varias páginas de notas, cada una con su propio título.
- **Resaltador de texto** — selecciona texto en una página y resáltalo. Cambia el color y el estilo, adjunta una nota y vuelve a ella más tarde.
- **Vuelve a encontrar tus resaltados** — los resaltados sobreviven a las recargas de página y se restauran en sitios dinámicos. En feeds largos la extensión sigue desplazándose y cargando contenido antiguo hasta encontrar el texto.
- **Gestor de notas guardadas** — explora todo lo que tienes a través de todos tus sitios, con búsqueda por sitio o por texto resaltado.
- **Copia de seguridad y restauración** — exporta todas las notas y ajustes a un archivo JSON e impórtalos en otro dispositivo.
- **Tres temas** — Green, Blue y Space.

## Por qué Overlayer

- **Totalmente sin conexión** — sin servidores, sin cuentas, sin analíticas, sin telemetría.
- **Privado por diseño** — tus notas nunca salen de tu dispositivo. Lee la [política de privacidad](PRIVACY.md).
- **Gratis y de código abierto** — GPL-3.0, sin planes de pago, sin anuncios, nada bloqueado tras una licencia.
- **Cualquier navegador Chromium** — Chrome, Edge, Opera, Brave, Vivaldi y más.

Mientras tanto, compílala desde el código fuente y carga la compilación sin empaquetar:

**Chrome / Edge** (y otros navegadores Chromium como Opera, Brave, Vivaldi)

1. `npm install && npm run build`
2. Abre `chrome://extensions` y activa el **modo de desarrollador**
3. Elige **Cargar descomprimida** y selecciona `.output/chrome-mv3`

## Desarrollo

Requiere Node.js 22 o superior.

```bash
npm install          # instalar dependencias
npm run dev          # Chrome con recarga en caliente
npm run check        # tipos, lint y pruebas
npm run build        # compilación de producción (también :edge)
npm run zip          # archivo empaquetado para enviar a la tienda
```

La calidad del código se garantiza con TypeScript en modo estricto, ESLint, Prettier y Vitest, más un análisis de Qodana en cada push.

## Tecnologías

Construido con [WXT](https://wxt.dev) sobre Manifest V3 para navegadores Chromium, React 19, TypeScript y Tailwind CSS 4. El texto enriquecido usa TipTap, y la interfaz usa primitivos de Radix UI con iconos de Lucide.

Las tres superficies — el popup, la página de ajustes y la interfaz en la página — están documentadas en el código junto con el modelo de almacenamiento y el enfoque de anclaje de resaltados. La interfaz en la página se monta dentro de shadow roots para que los estilos del sitio y de la extensión nunca colisionen.
