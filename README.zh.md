Overlayer 为每个网站提供专属的快速笔记面板，并让你直接在页面上高亮文本。你创建的一切都保存在浏览器里、在你的设备上 —— 没有服务器，没有账号，没有追踪。

## 功能

- **按网站的快速笔记** —— 在任意网站打开一个面板，添加富文本、便签、链接按钮和图片。可直接从电脑或从另一个页面拖入图片。
- **多个页面** —— 每个网站可以有多个笔记页面，每页都有自己的标题。
- **文本高亮** —— 在页面上选中文本并高亮。可更改颜色和样式、附上笔记，稍后再回到它。
- **重新找到你的高亮** —— 高亮在页面刷新后仍然存在，并在动态网站上恢复。在很长的信息流中，扩展会持续滚动并加载旧内容，直到找到该文本。
- **已保存笔记管理器** —— 浏览你所有网站上的全部内容，可按网站或按高亮文本搜索。
- **备份与恢复** —— 将所有笔记和设置导出为 JSON 文件，并在另一台设备上导入。
- **三种主题** —— Green、Blue 和 Space。

## 为什么选择 Overlayer

- **完全离线** —— 没有服务器，没有账号，没有分析统计，没有遥测。
- **隐私优先** —— 你的笔记永不离开你的设备。请阅读[隐私政策](PRIVACY.md)。
- **免费且开源** —— GPL-3.0，没有付费档位，没有广告，没有任何功能被锁在许可证之后。
- **所有 Chromium 浏览器** —— Chrome、Edge、Opera、Brave、Vivaldi 等。

## 开发

需要 Node.js 22 或更高版本。

```bash
npm install          # 安装依赖
npm run dev          # Chrome 热重载
npm run check        # 类型、lint 和测试
npm run build        # 生产构建（也支持 :edge）
npm run zip          # 用于提交商店的打包压缩文件
```

代码质量由严格模式的 TypeScript、ESLint、Prettier 和 Vitest 保障，并在每次 push 时运行 Qodana 扫描。

## 技术栈

基于 [WXT](https://wxt.dev) 构建，采用面向 Chromium 浏览器的 Manifest V3、React 19、TypeScript 和 Tailwind CSS 4。富文本由 TipTap 驱动，界面使用 Radix UI 原语与 Lucide 图标。

三个界面 —— 弹出窗口、设置页和页面内界面 —— 连同存储模型与高亮锚定方案一并记录在源码中。页面内界面挂载在 shadow root 内，使网站样式与扩展样式永不冲突。
