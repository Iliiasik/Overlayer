Overlayer даёт каждому сайту собственную шторку быстрых заметок и позволяет выделять текст прямо на странице. Всё, что вы создаёте, остаётся в браузере, на вашем устройстве — без серверов, без аккаунтов, без слежки.

## Возможности

- **Быстрые заметки на каждый сайт** — откройте шторку на любом сайте и добавляйте rich-text, стикеры, кнопки-ссылки и картинки. Перетаскивайте картинки прямо с компьютера или с другой страницы.
- **Несколько страниц** — у каждого сайта может быть несколько страниц заметок, у каждой свой заголовок.
- **Текстовый хайлайтер** — выделите текст на странице и подсветите его. Меняйте цвет и стиль, прикрепляйте заметку и возвращайтесь к ней позже.
- **Снова находит ваши выделения** — выделения переживают перезагрузку страницы и восстанавливаются на динамических сайтах. На длинных лентах расширение само прокручивает и подгружает старый контент, пока не найдёт текст.
- **Менеджер сохранённых заметок** — просматривайте всё по всем сайтам, с поиском по сайту или по выделенному тексту.
- **Бэкап и восстановление** — экспортируйте все заметки и настройки в JSON-файл и импортируйте их на другом устройстве.
- **Три темы** — Green, Blue и Space.

## Почему Overlayer

- **Полностью офлайн** — без серверов, без аккаунтов, без аналитики, без телеметрии.
- **Приватность по умолчанию** — ваши заметки не покидают устройство. Читайте [политику конфиденциальности](PRIVACY.md).
- **Бесплатно и с открытым кодом** — GPL-3.0, без платных тарифов, без рекламы, ничего не заперто за лицензией.
- **Любой браузер на Chromium** — Chrome, Edge, Opera, Brave, Vivaldi и другие.

А пока соберите из исходников и загрузите распакованную сборку:

**Chrome / Edge** (и другие браузеры на Chromium: Opera, Brave, Vivaldi)

1. `npm install && npm run build`
2. Откройте `chrome://extensions` и включите **режим разработчика**
3. Выберите **Загрузить распакованное расширение** и укажите `.output/chrome-mv3`

## Разработка

Требуется Node.js 22 или новее.

```bash
npm install          # установка зависимостей
npm run dev          # Chrome с горячей перезагрузкой
npm run check        # типы, линт и тесты
npm run build        # продакшен-сборка (также :edge)
npm run zip          # архив для подачи в магазин
```

Качество кода обеспечивают строгий TypeScript, ESLint, Prettier и Vitest, плюс скан Qodana на каждый push.

## Технологии

Собрано на [WXT](https://wxt.dev) под Manifest V3 для браузеров на Chromium, React 19, TypeScript и Tailwind CSS 4. Rich-text работает на TipTap, интерфейс использует примитивы Radix UI и иконки Lucide.

Три поверхности — попап, страница настроек и интерфейс на странице — задокументированы в исходниках вместе с моделью хранения и подходом к якорению выделений. Интерфейс на странице монтируется внутри shadow root, чтобы стили сайта и расширения никогда не конфликтовали.
